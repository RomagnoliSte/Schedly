# Schedly

### Gestão de agendamentos de serviços

Aplicação para organização e gestão de agendamentos de serviços

## Problema

Pequenos prestadores de serviço, costumam não gerir muito bem os agendamentos de serviços, em N setores, quando a demanda começa crescer.

## Solução

A aplicação permite cadastrar clientes, serviços e gerenciar os agendamentos.

## Funcionalidades

- Cadastro de serviços
- Cadastro de Clientes
- Cadastro de agendamentos
- Resumo com quantidade de agendamentos do dia, total de clientes, quantidade de serviços ativos e receita estimada do dia
- Lista de próximos agendamentos
- Filtro por status do agendamento
- Filtro por serviço

## Tecnologias

- React
- TypeScript
- Vite
- styled-components
- react-toastify
- react-icons

## Decisões de UX

- Interface limpa e com boa hierarquia visual
- Formulário com validação e labels claros
- Destaque visual para agendamentos confirmados
